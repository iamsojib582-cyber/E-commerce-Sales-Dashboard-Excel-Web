# E-commerce Sales Performance Dashboard

## Project Objective
To analyze e-commerce sales data to identify regional trends, customer demographics, and fulfillment efficiency for a clothing business in 2022.

## Key Features
* **Data Cleaning:** Handled 30,000+ rows, removed duplicates, and standardized categories.
* **Advanced Formulas:** Used `VLOOKUP` and `IFS` to categorize 29 states into 6 geographical regions.
* **Interactive Dashboard:** Built 8 dynamic charts connected to 3 slicers (Region, Channel, Status) for real-time data drilling.
